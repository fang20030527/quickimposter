# Describe and Vote 图标重绘设计

## 目标

重新绘制 Quick Imposter 原型中 `Describe and vote` 步骤图标，解决当前双气泡轮廓拥挤、图形黏连和小尺寸识别度不足的问题。

## 选定方向

采用用户选定的 B 方案：浅紫描边对话气泡与右上角圆形勾选徽章。

- 主体为单个圆角对话气泡，使用浅紫填充和深紫粗描边。
- 气泡内放置两条简化信息线，表达描述和讨论。
- 右上角叠放深紫圆形徽章，内部使用白色勾选符号，表达投票完成。
- 描边端点与转角保持圆润，与页面现有手机、盾牌和玩家图标一致。

## 交付范围

- 重绘 `assets/svg/step-chat-vote.svg`。
- 替换 `design.pen` 桌面端 `Describe and vote` 图标。
- 替换 `design.pen` 手机端 `Describe and vote` 图标。
- 保持现有图标位置、外框尺寸、步骤卡片、文案和布局不变。

## 尺寸与颜色

- SVG 使用 `viewBox="0 0 72 64"`。
- 桌面端显示尺寸保持 52×52。
- 手机端显示尺寸保持 48×48。
- 主描边使用深紫 `#2D0870` 或现有品牌深紫的等价值。
- 主体填充使用浅紫 `#EFE5FF`，徽章使用品牌紫 `#712BD0`，勾选使用白色。
- 两条信息线使用中亮度紫色，缩小后仍保持清晰间距。

## 验收标准

- 48px 和 52px 尺寸下均能清晰识别为“对话＋勾选”。
- 气泡、信息线、徽章和勾选之间没有黏连或意外重叠。
- 桌面端与手机端视觉一致，图标没有拉伸、裁切或偏移。
- `step-chat-vote.svg` 可独立打开并通过 XML 校验。
- Pencil 导出的桌面端和手机端预览均正常显示。

## 范围限制

- 不修改其他 SVG 图标、双手机插画、字体、文案或页面布局。
- 不修改 Next.js 网站代码。
