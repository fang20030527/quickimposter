# Quick Imposter

A mobile-first, single-device imposter word game built with Next.js 16, React 19, TypeScript, and Tailwind CSS 4.

## Development

This project uses pnpm exclusively.

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Verification

```bash
pnpm lint
pnpm test
pnpm build
```

Product requirements live in [`docs/PRD.md`](docs/PRD.md). The approved MVP design is in [`docs/superpowers/specs/2026-07-12-quick-imposter-core-mvp-design.md`](docs/superpowers/specs/2026-07-12-quick-imposter-core-mvp-design.md).
